# Social Links Profile - Frontend Mentor Challenge

This is a solution to the [Social Links Profile challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/social-links-profile-UG92b11w7). As a student in Computer Science (Tunisia), I focused on clean code and responsive layout.

## 🔗 Live Demo
[View the live project here](YOUR_GITHUB_PAGES_LINK_HERE)

## 🚀 Features
- **Mobile-First Design:** Optimized for devices like Samsung S8+ and modern punch-hole screens.
- **Custom Styling:** 60% CSS / 40% HTML ratio for a polished UI.
- **Interactive Elements:** Smooth hover states for all social links.
- **Clean Layout:** Used `inline-block` and `vertical-align` for precise element positioning.

## 🛠️ Built With
- **Semantic HTML5:** For better accessibility and SEO.
- **CSS Custom Properties:** Organized variables for colors and spacing.
- **Responsive Web Design:** Ensuring a seamless experience across all screen sizes.

## 🧠 What I Learned
During this project, I tackled several technical challenges:
1. **Inline-Block Spacing:** Solved the "white-space" gap issue between elements by managing font sizes and layout flow.
2. **Safe Area Padding:** Implemented top padding to ensure the UI doesn't conflict with modern mobile camera cutouts.
3. **Branding:** Applied a "Dark Academia" influenced aesthetic with custom border-radius and shadows.
